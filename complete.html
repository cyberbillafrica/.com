Certified Cybersecurity Analyst Course

From Zero to Certified in 7 Months

This course is designed to take learners from foundational cybersecurity knowledge to advanced skills in system protection, network security, incident response, penetration testing, and compliance.

Duration: 7 Months | Focus: CompTIA Security+ and PenTest Readiness

Month 1: Foundations of Cybersecurity & Security+ Basics

Week 1: Introduction to Cybersecurity
  • Introduction to cybersecurity
  • Course overview and career paths
  • Security terminology aligned with Security+
    • Weekly Hands-on practical session
Week 2: Core Security Concepts
  • Understanding confidentiality, integrity, availability (CIA triad)
  • Basics of threat actors and threat intelligence
  • Security awareness and best practices
    • Weekly Hands-on practical session
Week 3: Governance and Policies
  • Security policies, procedures, and governance basics
  • Emerging trends: Zero trust principles, secure DevOps basics
  • Introduction to social engineering
    • Weekly Hands-on practical session
Week 4: Virtualization and Linux Basics
  • Virtualization and Linux command line intro
  • System hardening basics
  • Setting up secure lab environments
    • Weekly Hands-on practical session: Setting up labs with VirtualBox, Linux basics, zero trust simulation

Monthly Bonus: Guided study in TryHackMe portal

Month 2: Networking & Network Security

Week 1: Networking Fundamentals
  • Networking fundamentals: OSI & TCP/IP models
  • IP addressing, subnetting, DNS, DHCP basics
  • Common protocols (TCP, UDP, HTTP, HTTPS, FTP, SMTP, SNMP)
    • Weekly Hands-on practical session
Week 2: Network Devices and Security
  • Network devices and their functions (routers, switches, firewalls)
  • Wireless network security and VPNs
  • Introduction to secure network architecture
    • Weekly Hands-on practical session
Week 3: Network Attacks
  • Network attacks (DoS/DDoS, spoofing, MITM)
  • Network security tools: Firewalls, IDS/IPS, NAC
  • Analyzing network vulnerabilities
    • Weekly Hands-on practical session
Week 4: Practical Network Security
  • Configuring secure network settings
  • Packet capture and analysis techniques
  • Simulating network defense scenarios
    • Weekly Hands-on practical session: Configure firewall rules, Wireshark packet capture and analysis, Cisco Packet Tracer lab

Month 3: Security Architecture & System Hardening

Week 1: Operating System Security
  • Operating system security (Windows, Linux)
  • User account management and permissions
  • Patch and vulnerability management
    • Weekly Hands-on practical session
Week 2: Endpoint and Application Security
  • Endpoint security: antivirus, anti-malware, EDR tools
  • Application security basics
  • Mobile device and IoT security
    • Weekly Hands-on practical session
Week 3: Cloud and Physical Security
  • Cloud security fundamentals and shared responsibility model
  • Physical security: Environmental controls, badge systems, locks, and surveillance
  • Securing cloud and physical environments
    • Weekly Hands-on practical session
Week 4: Data Protection
  • Data protection: Data classification, data loss prevention (DLP), secure data disposal
  • Implementing data protection measures
  • System hardening techniques
    • Weekly Hands-on practical session: Harden Windows & Linux systems, configure security settings, cloud lab intro, set up basic DLP policies, simulate physical access control

Month 4: Access Control, Identity Management & Cryptography

Week 1: Access Control Models
  • Access control models: DAC, MAC, RBAC, ABAC
  • Principles of access control
  • Authorization mechanisms
    • Weekly Hands-on practical session
Week 2: Authentication Methods
  • Authentication methods: MFA, biometrics, tokens
  • Identity and Access Management (IAM)
  • Configuring secure authentication
    • Weekly Hands-on practical session
Week 3: Cryptography Basics
  • Introduction to cryptography: symmetric, asymmetric encryption
  • Hashing, digital signatures, PKI, certificates
  • Secure protocols and their uses
    • Weekly Hands-on practical session
Week 4: Secure Communications
  • VPNs and secure communications (SSL/TLS, SSH)
  • Data protection in transit: Encryption for data in motion, secure file transfers
  • Implementing cryptographic solutions
    • Weekly Hands-on practical session: Configure IAM policies, use OpenSSL for encryption, create and verify digital signatures, set up secure file transfer with SFTP

  • Incident response lifecycle (Preparation to Lessons Learned)

    • Weekly Hands-on practical session

  • Digital forensics basics
  • Evidence collection and preservation
  • Forensic tools and techniques
    • Weekly Hands-on practical session

  • Risk management concepts: risk assessment, mitigation strategies

  • Risk mitigation planning
    • Weekly Hands-on practical session
Week 4: Business Continuity and Compliance
  • Business continuity planning (BCP) and disaster recovery planning (DRP)
  • Legal, compliance, and regulatory frameworks (GDPR, HIPAA, PCI-DSS)
  • Ethical considerations in cybersecurity
    • Weekly Hands-on practical session: Incident response simulation, forensic data capture, risk analysis exercise

Month 6: Penetration Testing & Vulnerability Assessment

Week 1: Penetration Testing Basics
  • Penetration testing methodologies and phases (planning, reconnaissance, exploitation, reporting)
  • Types of penetration testing (white-box, black-box, grey-box)
  • Introduction to penetration testing tools
    • Weekly Hands-on practical session
Week 2: Reconnaissance and Scanning
  • Reconnaissance and scanning tools: Nmap, Nessus, OpenVAS
  • Information gathering techniques
  • Vulnerability scanning basics
    • Weekly Hands-on practical session
Week 3: Exploitation and Attacks
  • Exploitation basics and post-exploitation techniques
  • Password attacks and brute force tools
  • Web application vulnerabilities (OWASP Top 10)
    • Weekly Hands-on practical session
Week 4: Advanced Penetration Testing
  • Wireless penetration testing basics
  • Social engineering techniques
  • Writing penetration testing reports
    • Weekly Hands-on practical session: Use Kali Linux tools, conduct vulnerability scans, perform basic exploit attempts, capture flag (CTF) style exercises
