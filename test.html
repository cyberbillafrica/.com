Certified Cybersecurity Analyst Course

From Zero to Certified in 7 Months

This course is designed to take learners from foundational cybersecurity knowledge to advanced skills. It provides a hands-on learning experience, covering the CompTIA Security+ and (ISC)² CC curriculums. It equally provides foundational preparation for CEH certifications. It generally equips you with job-ready skills in automation in system protection, network security, incident response, penetration testing, and compliance.

Duration: 7 Months | Focus: CompTIA Security+, (ISC)² CC and PenTest Readiness

  • Month 1- Security Principles
  • Month 2- Networking & Network Security
  • Month 3- Security Architecture
  • Month 4- Access Control Protocols
  • Month 5- Incident Response, DR & BC
  • Month 6- PenTest & Vulnerability Assessment
  • Month 7- Automation, SOC & Career
Month 1: Cybersecurity Fundamentals & Foundational Lab Setup
Week 1: Introduction to Cybersecurity
  • Overview of professional cybersecurity and career paths
  • Importance of cybersecurity in the digital world
  • Key terminology for Security+, ISC2 CC, and CEH
  • Hands-on: Lab Setup - Build Your Cyber Lab: Set up a home lab using VirtualBox/VMware with multiple VMs (Kali Linux, Ubuntu, Windows Server). Configure basic networking between them.
Week 2: Core Security Principles
  • Confidentiality, integrity, availability (CIA triad)
  • Threat actors, threat intelligence, and social engineering basics
  • Security awareness and best practices
  • Hands-on: Malware Analysis - Simulated Malware Analysis Lab: Use a controlled VM (e.g., FLARE VM or Ubuntu) to examine sample malware. Use strings, file, and hexdump to analyze file properties.
Week 3: Governance and Emerging Trends
  • Security policies, procedures, and governance
  • Zero trust principles and secure DevOps basics
  • Introduction to ethical hacking concepts (CEH foundation)
  • Hands-on: Awareness - Social Engineering Defense Lab: Role-play phishing scenarios using GoPhish to design mock phishing emails and analyze responses.
Week 4: Virtualization and Linux Basics
  • Virtualization and Linux command line fundamentals
  • System hardening techniques
  • Setting up secure lab environments
  • Hands-on: Policy - Mini-Assignment: Cybersecurity Policy Framework: Draft a security policy for a fictional small business using NIST templates.

Monthly Bonus: Guided study in TryHackMe portal

Month 2: Networking & Network Security
Week 1: Networking Fundamentals
  • OSI and TCP/IP models
  • IP addressing, subnetting, DNS, DHCP basics
  • Common protocols (TCP, UDP, HTTP, HTTPS, FTP, SMTP, SNMP)
  • Hands-on: Network Design - Subnetting & Secure Network Design: Use Cisco Packet Tracer to design a network with subnets, VLANs, and ACLs. Simulate ARP spoofing or DDoS.
Week 2: Network Security Tools
  • Network devices: routers, switches, firewalls
  • Wireless network security and VPNs
  • Network access control (NAC) principles
  • Hands-on: Network Defense - Firewall and IDS/IPS Configuration Lab: Set up pfSense, configure firewall rules, VPNs, and Snort/Suricata. Use Nmap to simulate attacks.
Week 3: Network Threats and Defenses
  • Network attacks (DoS/DDoS, spoofing, MITM)
  • Firewalls, IDS/IPS, and secure network architecture
  • Network reconnaissance (CEH foundation)
  • Hands-on: Analysis - Wireshark Packet Analysis Project: Capture live traffic, filter protocols, and identify attacks (e.g., DNS spoofing). Create a mitigation report.
Week 4: Practical Network Security
  • Configuring secure network settings
  • Packet capture and analysis for threat detection
  • Simulating network defense scenarios
  • Hands-on: Analysis - Continue Wireshark and Cisco Packet Tracer labs for advanced network defense simulation.

Monthly Bonus: Guided study in TryHackMe portal

Month 3: Security Architecture & System Hardening
Week 1: Operating System Security
  • Securing Windows and Linux operating systems
  • User account management and permissions
  • Patch and vulnerability management
  • Hands-on: OS Hardening - Harden a Windows and Linux System: Disable unnecessary services, enforce password policies, and audit accounts using Lynis and Microsoft Defender.
Week 2: Endpoint and Application Security
  • Endpoint security: antivirus, anti-malware, EDR tools
  • Application security basics
  • Mobile device and IoT security
  • Hands-on: Endpoint Protection - Endpoint Protection Lab: Install and configure Microsoft Defender on a Windows VM. Simulate malware injection and observe detection.
Week 3: Cloud and Physical Security
  • Cloud security fundamentals and shared responsibility model
  • Physical security: environmental controls, badge systems, surveillance
  • Securing cloud and physical environments
  • Hands-on: Application Hardening - Secure Ubuntu Web Server: Install Apache/Nginx, enable UFW, and implement SSL/TLS (Let’s Encrypt principles).
Week 4: Data Protection
  • Data classification, data loss prevention (DLP), secure data disposal
  • Implementing data protection measures
  • Advanced system hardening techniques
  • Hands-on: Policy/Audit - Patch Management & CIS Benchmarks: Script a Bash/PowerShell tool to check/apply patches and assess VMs using CIS Benchmarks.

Monthly Bonus: Guided study in TryHackMe portal

Month 4: IAM, Cryptography & Secure Protocols
Week 1: Access Control Models
  • Access control models: DAC, MAC, RBAC, ABAC
  • Principles of access control
  • Authorization mechanisms (ISC2 CC foundation)
  • Hands-on: Access Control - IAM Project: Configure RBAC in Linux using users, groups, and sudo. Set up MFA on a sample application.
Week 2: Authentication Methods
  • Authentication methods: MFA, biometrics, tokens
  • Identity and Access Management (IAM)
  • Configuring secure authentication
  • Hands-on: Access Control - Continue IAM project, testing MFA bypass attempts in a controlled environment.
Week 3: Cryptography Fundamentals
  • Symmetric and asymmetric encryption
  • Hashing, digital signatures, PKI, certificates
  • Secure protocols and their applications
  • Hands-on: Cryptography - PKI Certificate Management Project: Use OpenSSL to generate a CA, issue certificates, and set up a secure HTTPS server.
Week 4: Secure Communications
  • VPNs and secure communications (SSL/TLS, SSH)
  • Data protection in transit: encryption, secure file transfers
  • Implementing cryptographic solutions (CEH foundation)
  • Hands-on: Protocols - Secure Protocol Testing Lab: Configure SSH and VPN tunnels, inspect encrypted traffic with Wireshark.

Monthly Bonus: Guided study in TryHackMe portal

Month 5: Incident Response, Risk Management & Compliance
Week 1: Incident Response Fundamentals
  • Incident response lifecycle (Preparation to Lessons Learned)
  • Types of incidents and handling techniques
  • Incident detection and analysis (ISC2 CC foundation)
  • Hands-on: Incident Response - Incident Response Simulation: Inject a controlled breach (e.g., malware or unauthorized login) and investigate logs.
Week 2: Digital Forensics
  • Digital forensics basics
  • Evidence collection and preservation
  • Forensic tools and techniques (CEH foundation)
  • Hands-on: Digital Forensics - Digital Forensics Lab: Use Autopsy or Sleuth Kit on a disk image to recover files and analyze artifacts.
Week 3: Risk Management
  • Risk assessment and mitigation strategies
  • Identifying and prioritizing risks
  • Risk management frameworks
  • Hands-on: Risk/Compliance - Compliance Audit & Risk Management Project: Assess a mock environment against GDPR/HIPAA/PCI-DSS using OpenVAS/Nessus.
Week 4: Business Continuity and Compliance
  • Business continuity planning (BCP) and disaster recovery planning (DRP)
  • Legal, compliance, and regulatory frameworks (GDPR, HIPAA, PCI-DSS)
  • Ethical considerations in cybersecurity
  • Hands-on: Risk/Compliance - Continue compliance audit, create a prioritized mitigation report.

Monthly Bonus: Guided study in TryHackMe portal

Month 6: Penetration Testing & Vulnerability Assessment
Week 1: Penetration Testing Basics
  • Penetration testing methodologies and phases
  • Types of penetration testing (white-box, black-box, grey-box)
  • Introduction to pen testing tools (CEH foundation)
  • Hands-on: Network Pentest - Network Penetration Testing Project: Use Nmap and Nessus/OpenVAS to map a lab network and identify vulnerabilities.
Week 2: Reconnaissance and Scanning
  • Reconnaissance and scanning tools: Nmap, Nessus, OpenVAS
  • Information gathering techniques
  • Vulnerability scanning basics
  • Hands-on: Network Pentest - Continue network pentest, using Metasploit for exploitation in a controlled environment.
Week 3: Exploitation Techniques
  • Exploitation basics and post-exploitation techniques
  • Password attacks and brute force tools
  • Web application vulnerabilities (OWASP Top 10)
  • Hands-on: Web App Pentest - Web Application Vulnerability Scanning Lab: Use Burp Suite and Nikto on OWASP Broken Web Apps or VulnHub.
Week 4: Advanced Penetration Testing
  • Wireless penetration testing basics
  • Social engineering techniques
  • Writing penetration testing reports
  • Hands-on: Applied Offense - CTF Challenge Creation/Solving: Design or solve a CTF using CTFd or a public platform.

Monthly Bonus: Guided study in TryHackMe portal

Month 7: Automation, SOC Tools & Career Preparation
Week 1: Scripting for Cybersecurity
  • Python for cybersecurity automation
  • PowerShell and Bash scripting basics
  • Scripting for security tasks (CEH foundation)
  • Hands-on: Automation - Python Automation Script: Develop a script for log parsing or a port scanner using Scapy or Pandas.
Week 2: Automation and Version Control
  • Automating repetitive security tasks
  • Using Git and GitHub for version control
  • Scripting for incident response
  • Hands-on: Automation - Continue automation script development, integrating with Git for version control.
Week 3: Security Operations Center (SOC)
  • Security monitoring and SOC fundamentals
  • Introduction to SIEM tools
  • Analyzing security logs and alerts (ISC2 CC foundation)
  • Hands-on: SIEM/SOC - SIEM Threat Hunting Lab: Set up Splunk, ELK Stack, or Wazuh to ingest logs and hunt for IOCs.
Week 4: Career Preparation and Capstone
  • Cybersecurity career guidance and job preparation
  • Mock interviews, resume building, certifications guidance
  • Final capstone project: Simulate an incident, perform assessment, and respond with mitigation plan
  • Hands-on: Capstone - Full SOC Attack & Response Simulation: Red Team simulates a breach, Blue Team responds using automation and SIEM tools.
  • Hands-on: Career Prep - Portfolio Creation: Document projects in a GitHub repo or portfolio website, including mock Security+ tests.

Monthly Bonus: Guided study in TryHackMe portal
